    在工程里启用这个脚本后，建立新事件。随便选择一个角色作为图片，然后建立plugin command，在文本框内输入(大小写无所谓)：

SimpleText Setup 0 Duration 2

    SimpleText是这个插件的主识别代码。Setup提示程序这条代码是用来设置文字的。0是文字绑定的事件，可以用事件ID或者事件名（就是事件窗口左上角那个。注意：不能有空格）。为了省事，可以用0代表当前事件，用-1代表主角。这里的0指的就是正在编辑的事件。Duration告诉程序这次修改的是文字的显示时间。2是两秒。也就是说，这条命令让在此事件产生的所有文字维持两秒。

    在刚才建立的plug command下面新建一个plugin command，在文本框内输入:

SimpleText Setup 0 SlideUp true

    这条命令告诉程序：让文字向上方滑行。

    重复以上过程，输入：

SimpleText Setup 0 Color #ff0000

    这条命令把文字设定成红色。颜色代码可以在网上查。

    最后，输入：

SimpleText Write 0 -99999

    注意这次是Write而不是Setup。这条命令在次事件上方（位置可以用其他命令修改，具体命令看说明）显示“-99999”的文字。显而易见，文字是红色的，并会在产生的同时向上滑行两秒钟，然后消失。就像某些游戏中单位受伤害时那样。滑行速度当然也可以通过其他命令修改。

以上是四个例子中的一个。可以试试另外几个。把命令一句句复制进plugin command就行了。
